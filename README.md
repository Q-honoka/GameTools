# GameTools
C++でゲームを作るときに使えるツールをまとめたリポジトリ

# Vector2D
## 概要
２次元座標や方向、速度を扱うための構造体です。

## できること
- 四則演算
- 正規化
- 長さ取得
- 内積
- 距離計算
- 比較

# TimeManager
## 概要
ゲーム内の時間やFPS管理をするクラスです。

## できること
- FPSの設定
- デルタタイムの取得
- フレームカウントの取得
- ゲーム開始からの経過時間の取得

# Collision2D
## 概要
2次元空間での当たり判定を行う関数集です。
Shape2Dで定義した形を表す構造体を使用しています。

## できること
- 点と点の衝突判定
- 点と線分の衝突判定
- 点と円の衝突判定
- 点と矩形の衝突判定
- 円と円の衝突判定
- 円と矩形の衝突判定
- 矩形と矩形の衝突判定
